# Drupal-Test

* A suite of scripts to test UofC Drupal sites.

## Dependencies

* [Composer](https://getcomposer.org)
* PHP 5.35+

## Installation

* Brew
	
		ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"

* Composer

		brew install composer
		
* App dependencies

		composer install
